# Example: TwiBot-20

This folder contains a public example preview generated with the `academic-ppt` skill.

## Source

- Local input used during generation: `TwiBot-20.pdf`
- Source path is intentionally omitted from this public repository.
- The original paper PDF is not included in this repository.

## Included Artifacts

- `montage.png`: 16-slide rendered montage for quick visual inspection
- `cover.png`: cover slide preview

## Why This Example Exists

This example demonstrates that the skill can:

- normalize a research PDF into a working brief
- plan an academic presentation structure automatically
- build an editable PowerPoint deck
- render and validate the deck successfully

## Validation Snapshot

The example deck was generated and validated locally with:

- rendered slide previews
- overflow check passed
- font check passed

This example is intentionally lightweight. The generated `.pptx`, temporary workspace, and local source PDF are not committed.
