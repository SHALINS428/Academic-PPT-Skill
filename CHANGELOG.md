# Changelog

All notable changes to this repository should be documented in this file.

The format is based on Keep a Changelog principles and this repository currently uses a simple manual versioning approach.

## [0.1.0] - 2026-05-13

### Added

- initial public release of the `academic-ppt` skill
- editable PowerPoint generation workflow based on JavaScript authoring
- source normalization pipeline for `.doc`, `.docx`, `.pdf`, `.md`, `.pptx`, and images
- deck planning workflow for thesis-defense narratives
- validation scripts for render checks, overflow checks, and font checks
- MIT license
- third-party attribution file for upstream open-source references
- repository-level README and contribution guidance

### Notes

- this release focuses on thesis-defense and academic-report scenarios rather than general business slide generation
- the repository includes adapted assets and workflow ideas attributed in `THIRD_PARTY_NOTICES.md`
